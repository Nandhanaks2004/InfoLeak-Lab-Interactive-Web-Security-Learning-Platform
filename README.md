# 🔐 Info Disclosure Lab

### Interactive Web Learning Module for Information Disclosure

---

## 🚀 Overview

This project is a **dark-themed interactive web module** designed to help students understand **Information Disclosure vulnerabilities** in a practical and visual way.

It is a **single-page HTML learning platform** — no installation, no setup.

👉 Just open the file and start learning.

---

## 🎯 What This Module Teaches

* What Information Disclosure is
* How attackers think and use leaked data
* Common developer mistakes (misconfigurations)
* Real-world attack scenarios
* Step-by-step vulnerability exploitation
* How to fix vulnerabilities like a pentester

---

## 🧠 Learning Experience

This is not just notes — it is designed like a **mini cybersecurity platform**:

* 🔎 Interactive simulations (think like an attacker)
* 🧪 Real attack scenarios
* 📘 Structured theory (beginner → advanced)
* 🎯 Guided lab walkthroughs
* 🛡️ Fix and mitigation explanations

---

## 🧪 Labs Included

* Information Disclosure via Error Messages
* Debug Page Information Leak
* Source Code Disclosure via Backup Files
* Authentication Bypass via Header Manipulation

Each lab includes:

* Goal
* Step-by-step solution
* Explanation of *why it works*
* Attacker mindset
* Fix recommendations
